About Task Manager

Task Manager is a web application designed to help users plan, track, and manage their tasks efficiently.

How to Get Started

  • Register: Create an account through the Register page.
  • Login: Log in to your account via the Login page.
  • Create tasks: Navigate to the Dashboard and start creating tasks.

Key Features

  • Task filtering and prioritization.
  • Status updates (new, in progress, completed).
  • Priority management for better task organization.

Contact Us

If you have any questions, reach out at support@taskmanager.com.
